# Launch Checklist

## Required
- [ ] Add `assets/images/hero.jpg`
- [ ] Replace Shopify URL in `assets/js/config.js`
- [ ] Confirm Etsy and Instagram URLs
- [ ] Proofread brand voice
- [ ] Test every CTA on mobile
- [ ] Upload repository contents to `main`
- [ ] Enable GitHub Pages from `main` / root
- [ ] Add custom domain in GitHub before changing DNS
- [ ] Configure apex and www DNS
- [ ] Enable HTTPS
- [ ] Submit sitemap to Google Search Console

## Commerce
- [ ] Link guide articles to specific products, not only store homepages
- [ ] Use UTM reporting in Shopify/Etsy analytics
- [ ] Add returns, shipping and product policies to the store platforms
- [ ] Confirm all product claims match actual materials and fulfillment

## Content
- [ ] Add one original image to each major article
- [ ] Add firsthand notes after each real visit
- [ ] Publish 1 useful guide and 2–4 supporting Instagram posts per week
- [ ] Refresh rules, hours and seasonal details quarterly
- [ ] Create a dedicated landing page for every proven high-converting product cluster
